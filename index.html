Privacy Policy for YT Don't BS Me Extension

Last Updated: [Insert Date]

1. Information Collection

This extension does not collect, store, or transmit any personal user data or browsing activity. The only information stored locally is:

  • Gemini API key (if provided by user)
  • Supadata API key (if provided by user)
  • User preferences/settings for the extension

2. Data Storage

All stored information remains exclusively on the user's device in Chrome's synchronized storage. No data is sent to any external servers except:

  • YouTube URLs sent to Supadata for transcription
  • Transcripts sent to Gemini API for analysis

These are processed but not stored by the extension.

3. Data Usage

API keys are used solely to authenticate with their respective services to provide the extension's functionality. They are not used for any other purpose.

4. Data Sharing

No user data is shared with any third parties except as necessary for the core functionality (sending to Supadata and Gemini APIs as described above).

5. Security

While we implement reasonable security practices, users should treat their API keys as sensitive information since they provide access to paid services.

6. Changes to This Policy

Any changes will be posted here with updated dates. Continued use constitutes acceptance.
